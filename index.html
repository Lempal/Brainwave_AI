Sobre o BrainWave

      

        Este projeto utiliza a API gratuita da Hugging Face para gerar textos com modelos de IA.
        É um exemplo intermediário usando HTML5, CSS3, JS e Python (FastAPI).
      

      

        Você pode modificar o backend para trocar o modelo e ajustar o comportamento da IA.
